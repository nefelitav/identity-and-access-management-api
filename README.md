# Auth API

## Run server

```
docker-compose up -d
```

## Services

- Admin:
  - get users
  - get user
  - delete users
  - delete user
  - update user
- Auth:
  - register
  - login
  - logout
  - refresh token
- Captcha
  - verify token
- MFA
  - OTP request code with email
  - OTP request code with sms
  - OTP verify code
  - TOTP enable
  - TOTP disable
  - TOTP confirm & enable
  - TOTP verify
- Profile
  - get user
  - update user
  - delete user
  - reset password
  - request password reset
